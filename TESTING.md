# 🧪 מדריך בדיקות - Sales Coach AI

## 🚀 הפעלת הדמו

### שיטה 1: סקריפט אוטומטי (מומלץ)

```bash
./start-demo.sh
```

### שיטה 2: ידני

```bash
cd demo
python3 -m http.server 8080
```

השרת יעלה על: **http://localhost:8080**

## ✅ רשימת בדיקות

### 1. בדיקת טעינת האקסטension

- [ ] פתח `chrome://extensions/`
- [ ] ודא ש-"Developer mode" מופעל
- [ ] לחץ "Load unpacked" ובחר את תיקיית `dist`
- [ ] ודא שהאקסטension מופיע ברשימה ללא שגיאות
- [ ] ודא שהאייקון מופיע בסרגל הכלים

**✅ הצלחה:** האקסטension טעון ללא שגיאות אדומות

---

### 2. בדיקת הגדרות API Key

- [ ] לחץ על אייקון האקסטension בסרגל הכלים
- [ ] לחץ על ⚙️ Settings
- [ ] הזן API Key (OpenAI או Anthropic)
- [ ] בחר מודל (GPT-4 Turbo / Claude 3 Opus)
- [ ] לחץ "Save Settings"
- [ ] ודא שההגדרות נשמרו

**✅ הצלחה:** ההגדרות נשמרות ומופיעות בפעם הבאה

---

### 3. בדיקת הדמו - תמלול

- [ ] דבר למיקרופון
- [ ] בדוק שהתמלול מופיע באזור "תמלול בזמן אמת"

**✅ הצלחה:** התמלול מופיע בזמן אמת

### 4. בדיקת הדמו - הצעות AI

- [ ] המשך לדבר למיקרופון

**✅ הצלחה:** הצעות AI מופיעות אחרי כמה שניות

## 🐛 פתרון בעיות נפוצות

### השרת לא עולה

```bash
# בדוק אם הפורט תפוס
lsof -ti:8080

# הרוג תהליך קיים
lsof -ti:8080 | xargs kill -9

# נסה שוב
cd demo && python3 -m http.server 8080
```

### האקסטension לא טעון

- ודא שבחרת את תיקיית `dist` (לא `extension`)
- ודא ש-"Developer mode" מופעל
- בדוק את ה-Console ב-`chrome://extensions/` לשגיאות

### התמלול לא עובד

- ודא שהדפדפן תומך ב-Web Speech API (Chrome/Edge)
- בדוק שהמיקרופון עובד
- בדוק את ה-Console (F12) לשגיאות
- ודא שהשפה מוגדרת נכון (he-IL)

### הצעות AI לא מופיעות

- ודא שה-API Key מוגדר נכון
- בדוק שיש קרדיטים בחשבון ה-API
- פתח את ה-Console (F12) ובדוק שגיאות רשת
- בדוק את טאב Network (F12) לראות קריאות API

### שגיאות Console

- **"API key invalid"** → בדוק את ה-API Key
- **"Rate limit exceeded"** → המתן או שדרג את התוכנית
- **"Network error"** → בדוק את החיבור לאינטרנט
- **"Permission denied"** → בדוק הרשאות מיקרופון
