## 🚀 איך להתחיל (5 שלבים פשוטים)

### שלב 1: טען מחדש את התוסף ✅
```
1. פתח Chrome
2. עבור ל: chrome://extensions/
3. מצא "Sales Coach AI"
4. לחץ על כפתור 🔄 (Reload) ליד התוסף
```

### שלב 2: רענן את דף הפגישה ✅
```
1. חזור לפגישה (Zoom/Meet/Teams)
2. לחץ F5 או רענן את הדף
```

### שלב 3: פתח Console כדי לראות מה קורה ✅
```
1. לחץ F12
2. עבור לטאב "Console"
3. השאר את זה פתוח כדי לראות את הלוגים
```

### שלב 4: התחל אימון! 🎤
```
1. לחץ על "Start Coaching" בפינה הימנית העליונה
2. Chrome יבקש הרשאה למיקרופון - אשר!
3. תראה הודעה: "✅ Speech recognition started"
```

### שלב 5: התחל לדבר עם הלקוח! 🗣️
```
אתה: "שלום, תודה שהתפנית לפגישה"
לקוח: "שלום, בכיף. ספר לי על הפתרון"
```

## 🔍 איך לבדוק שזה עובד:

### ✅ רשימת בדיקות:

1. **התוסף טעון?**
   - עבור ל: `chrome://extensions/`
   - וודא שהתוסף פעיל (כחול)

2. **הפאנל מופיע?**
   - תראה "Sales Coach AI" בפינה ימנית עליונה

3. **Speech Recognition עובד?**
   - פתח Console (F12)
   - לחץ "Start Coaching"
   - תראה: "✅ Speech recognition started"

4. **התמלול עובד?**
   - דבר משהו בעברית
   - תראה ב-Console: "🎤 FINAL: ..." או "🎤 interim: ..."
   - תראה על המסך את התמלול

5. **זיהוי דוברים עובד?**
   - תראה ב-Console: "👤 Speaker detected as: CLIENT" או "SALESPERSON"

6. **המאמן AI עובד?**
   - אחרי שהלקוח מדבר, תראה:
     "🤖 CLIENT SPOKE! Generating AI coaching suggestion..."
   - אחרי 5-10 שניות, יופיע כרטיס המאמן

## 🐛 פתרון בעיות:

### בעיה: לא רואה תמלול

**פתרון:**
```
1. בדוק ב-Console אם יש שגיאות אדומות
2. בדוק שהמיקרופון עובד:
   - הגדרות Chrome → פרטיות ואבטחה → הגדרות אתר → מיקרופון
   - וודא שהדף שלך מורשה
3. רענן את הדף (F5) ונסה שוב
```

### בעיה: זיהוי דוברים לא מדויק

**פתרון:**
```
המערכת משתמשת במילות מפתח ובהתחלפות דוברים.
אתה יכול לעזור לה על ידי:

1. שימוש במילים ייחודיות:
   - אתה: "אנחנו יכולים לתת לך..."
   - לקוח: "אני צריך להבין..."

2. אם זה מזהה לא נכון, זה לא נורא!
   המערכת תמשיך לעבוד ותתאים את עצמה
```

### בעיה: ה-AI לא מגיב

**בדוק:**
```
1. האם המפתח של OpenAI מוגדר?
   - לחץ ימני על התוסף → Options
   - בדוק שיש OpenAI API Key

2. האם יש קרדיט ב-OpenAI?
   - עבור ל: https://platform.openai.com/usage
   - בדוק שיש יתרה

3. בדוק ב-Console שגיאות:
   - חפש "❌" אדום
   - אם יש שגיאת API, תראה את הסיבה
```

### בעיה: הכרטיס לא מופיע

**בדוק:**
```
1. ב-Console, האם יש:
   "🤖 CLIENT SPOKE! Generating AI coaching suggestion..."?

2. אם כן, המתן 5-10 שניות - ה-AI צריך זמן

3. אם עדיין לא, בדוק אם יש שגיאה:
   "❌ AI streaming error: ..."
```

## 🎯 טיפים לשימוש מוצלח:

### 1. השתמש בכפתור "העתק" 📋

- 🔴 אדום = עצור! משהו דורש טיפול מיידי

תמיד תדע בדיוק מה לעשות, גם אם הלקוח הפתיע אותך.
