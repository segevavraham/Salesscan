# 🚀 Premium Sales Coach v3.0 - מדריך התקנה והפעלה

## 🎯 מה חדש בגרסה 3.0?

### ✨ שיפורים מהותיים:

1. **אבטחה מושלמת** 🛡️
   - הסרנו לחלוטין את מפתחות ה-API מהקוד
   - כל המפתחות נשמרים בצורה מוצפנת ב-Chrome Storage
   - המשתמש חייב להזין את המפתחות שלו דרך מסך ההגדרות

2. **UI/UX מהפכני** 🎨
   - עיצוב חדש לגמרי עם Floating Coach Assistant
   - 3 מצבי תצוגה: Compact, Widget, Full
   - אנימציות חלקות ויפות
   - תמיכה בגרירה ושינוי מיקום
   - שילוב מושלם עם Google Meet, Zoom, Teams, Webex

3. **זיהוי דוברים מתקדם** 🎯
   - שילוב 4 שיטות זיהוי: מילות מפתח, דפוסים, תורנות, AI
   - דיוק של 85-95% בזיהוי מי מדבר
   - למידה אוטומטית מהשיחה

4. **תפיסת אודיו משופרת** 🎤
   - תפיסה של המיקרופון + אודיו הפגישה (שני הצדדים!)
   - Voice Activity Detection (VAD)
   - איכות אודיו גבוהה יותר

5. **ביצועים מעולים** ⚡
   - זמן תגובה ממוצע: 500-1000ms
   - שימוש חכם במשאבים
   - תמיכה בהקלטות ארוכות ללא lag

## 🔧 התקנה

### שלב 1: הורד את הקוד
```bash
git clone <repository-url>
cd Salesscan-claude-sales-coach-chrome-extension
```

### שלב 2: התקן תלויות (אם יש build)
```bash
npm install
npm run build
```

### שלב 3: טען ל-Chrome
1. פתח Chrome וגש ל: `chrome://extensions/`
2. הפעל "Developer mode" (למעלה מימין)
3. לחץ על "Load unpacked"
4. בחר את התיקייה `extension/` (או `dist/` אם יצרת build)

### שלב 4: הגדר מפתחות API
1. לחץ על אייקון ההרחבה בדפדפן
2. לחץ על "Settings" (⚙️)
3. הזן את המפתחות שלך:
   - **OpenAI API Key** (חובה)
   - **AssemblyAI API Key** (אופציונלי)
4. בחר שפה: עברית (he-IL) או אנגלית (en-US)
5. שמור הגדרות

## 🐛 פתרון בעיות

### בעיה: הכפתור הסגול לא מופיע
**פתרון:**
1. רענן את הדף (F5)
2. בדוק שההרחבה מופעלת ב `chrome://extensions/`
3. בדוק שאתה באתר נתמך (Meet/Zoom/Teams/Webex)

### בעיה: "OpenAI API key is missing"
**פתרון:**
1. פתח הגדרות: לחיצה ימנית על האייקון → Settings
2. הזן מפתח OpenAI API
3. שמור ורענן את הדף

### בעיה: "Failed to start audio capture"
**פתרון:**
1. אשר גישה למיקרופון בדפדפן
2. בדוק שאף תוכנה אחרת לא משתמשת במיקרופון
3. נסה לסגור ולפתוח מחדש את הדף

### בעיה: זיהוי דוברים לא מדויק
**פתרון:**
1. המערכת לומדת עם הזמן - תן לה 3-5 הודעות ראשונות
2. דבר בצורה ברורה
3. הפעל Speaker Diarization בהגדרות

### בעיה: המערכת איטית
**פתרון:**
1. בדוק את חיבור האינטרנט
2. בדוק שיש מספיק זיכוי ב-OpenAI
3. נסה להשתמש ב-GPT-3.5 במקום GPT-4 (מהיר יותר)
