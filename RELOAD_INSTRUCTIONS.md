## 🐛 פתרון בעיות

### בעיה: לא רואה כפתור סגול

**פתרון:**
```
1. לחץ F12 בדף הפגישה
2. Console → בדוק אם יש שגיאות אדומות
3. אם יש שגיאת "Cannot find module":
   - זה אומר שהקבצים לא נטענו
   - עשה Remove + Load unpacked שוב
```

### בעיה: רואה את ה-UI הישן (לא את הכפתור הסגול)

**פתרון:**
```
זה אומר ש-Chrome טען את הגרשה הישנה מ-cache:

1. Shift + F5 (hard refresh) בדף הפגישה
2. אם לא עוזר:
   - chrome://extensions/
   - Remove את ההרחבה
   - Restart Chrome לגמרי
   - Load unpacked מחדש
```

### בעיה: Console מראה שגיאות

**שגיאות נפוצות:**

#### שגיאה: "Failed to load module"
```
פתרון:
1. ודא שכל הקבצים קיימים:
   extension/components/floating-coach-assistant.js
   extension/content/premium-sales-coach.js
   extension/services/web-speech-recognition.js

2. אם חסרים - עשה git pull
```

#### שגיאה: "OpenAI API key is missing"
```
פתרון:
1. לחץ על אייקון ההרחבה
2. לחץ ימני → Options
3. הזן מפתח OpenAI
4. שמור
5. רענן את דף הפגישה
```

## 🔍 בדיקה מהירה שהכל תקין

הרץ את הפקודות האלה:

```bash
# 1. בדוק שהקבצים קיימים
ls -lh extension/components/floating-coach-assistant.js
ls -lh extension/content/premium-sales-coach.js

# 2. בדוק את ה-manifest
cat extension/manifest.json | grep "premium-sales-coach"

# 3. בדוק את הגרסה
cat extension/manifest.json | grep version

# 4. בדוק את ה-service worker
cat extension/background/service-worker.js | grep "premium-sales-coach"
```

אם כל הפקודות עובדות בלי שגיאות - הכל תקין!
