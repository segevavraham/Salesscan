# 🔍 בדיקה מהירה - Sales Coach AI

## מה ליצרתי לך:

יצרתי **דף בדיקה מלא** שבודק את כל הפונקציות צעד אחר צעד.

---

## 📋 איך לבדוק (5 דקות):

### שלב 1: טען את התוסף
```
1. chrome://extensions/
2. לחץ 🔄 Reload על "Sales Coach AI"
3. וודא שהתוסף פעיל (כחול)
```

### שלב 2: פתח את דף הבדיקה
```
1. פתח קובץ חדש:
   File → Open File

2. בחר את הקובץ:
   test-page.html
   (בתיקיית הפרויקט)

3. הדף ייפתח בדפדפן
```

### שלב 3: בדוק זיהוי דיבור
```
1. לחץ "התחל זיהוי דיבור"

2. אשר גישה למיקרופון

3. דבר בעברית:
   - "שלום, אני רוצה לדבר על הפתרון"
   - "המחיר נשמע יקר מדי"
   - "אני צריך לחשוב על זה"

4. בדוק שאתה רואה:
   ✅ לוג עם התמלול
   ✅ תמלול מופיע בתיבה למטה
   ✅ נורה ירוקה ליד "זיהוי דיבור"
```

### שלב 4: בדוק את התוסף בפגישה אמיתית
```
1. היכנס לפגישה (Zoom/Meet) או פשוט google.com

2. תראה פאנל "Sales Coach AI" בפינה ימנית

3. פתח Console (F12)

4. לחץ "Start Coaching"

5. דבר ובדוק Console - מה כתוב שם?
```

---

## ❌ מה לעשות אם משהו לא עובד:

### בעיה: לא רואה תמלול בדף הבדיקה

**פתרון:**
1. בדוק ב-Console (F12) אם יש שגיאות
2. בדוק שהמיקרופון מורשה:
   chrome://settings/content/microphone
3. נסה דפדפן אחר (Chrome עובד הכי טוב)

### בעיה: התוסף לא מופיע בדף הבדיקה

**זה בסדר!** דף הבדיקה בודק את זיהוי הדיבור בלבד.
התוסף אמור להופיע רק בדפי Zoom/Meet/Teams/google.com

### בעיה: התמלול לא מדויק

**זה תלוי באיכות המיקרופון והרעש ברקע.**
- דבר בקול רם וברור
- הקטן רעשי רקע
- דבר קרוב למיקרופון
