# 🚀 התחלה מהירה - Sales Coach AI

## ✅ המערכת מוכנה לשימוש מיידי!

**כל מה שצריך כבר מוגדר:**
- ✅ ElevenLabs API (תמלול בעברית)
- ✅ OpenAI GPT-4 Turbo API (מאמן מכירות)
- ✅ UI בעברית מקצועי
- ✅ התראות חכמות בזמן-אמת

---

## 📋 3 שלבים פשוטים להתחלה

### 1️⃣ טען את התוסף ב-Chrome (30 שניות)

1. פתח Chrome
2. עבור ל: `chrome://extensions/`
3. הפעל **"Developer mode"** (מתג למעלה מימין)
4. לחץ **"Load unpacked"**
5. בחר את תיקיית **`dist/`**
6. ✅ התוסף יופיע ברשימה

---

### 2️⃣ היכנס לפגישה

1. היכנס לפגישת **Zoom** / **Google Meet** / **Microsoft Teams**
   (או פשוט **google.com** לבדיקה מהירה)
2. ✅ תראה פאנל "Sales Coach AI" בפינה הימנית העליונה

---

### 3️⃣ התחל אימון!

1. לחץ על **"Start Coaching"**
2. אשר גישה למיקרופון
3. **התחל לדבר!**
4. ✅ תמלול יופיע תוך שניות
5. ✅ המלצות AI יופיעו תוך 5-10 שניות

**זהו! אתה מקבל אימון בזמן-אמת** 🎯

---

## 🎯 מה תראה על המסך

### 1. תמלול חי בעברית 📝
```
[09:45:32] אתה: "שלום, תודה שהתפנית לפגישה"
[09:45:38] לקוח: "שלום, בכיף. ספר לי יותר על הפתרון"
```

### 2. התראות מיידיות צבעוניות 🚨

**🟢 ירוק - אות קנייה:**
```
🎯 הלקוח שאל על תהליך ההטמעה!
   זה אות קנייה חזק - קדם את השיחה!
```

**🟡 כתום - התנגדות:**
```
⚠️ הלקוח אמר "יקר מדי"
   זמן לדבר על ערך, לא על מחיר
```

**🔴 אדום - סיכון:**
```
🚨 הלקוח נשמע לא משוכנע
   עצור ובדוק מה מפריע לו
```

### 3. כרטיס המאמן המלא 💎

תקבל:
- **ניתוח מהיר**: סנטימנט, מוכנות להחלטה, רמת מעורבות
- **💭 מה הלקוח חושב**: ניתוח פסיכולוגי של מצב הלקוח
- **⭐ התגובה המומלצת**: תשובה מוכנה עם כפתור העתקה
- **🎭 3 אפשרויות נוספות**: תשובות חלופיות
- **🔍 שאלות מומלצות**: מה כדאי לשאול עכשיו
- **📝 הערות המאמן**: מה אתה עושה טוב, מה לשפר, מה להימנע
- **🎯 צעדים הבאים**: מה לעשות עכשיו, בקצר טווח, ואיך לסגור

## ⌨️ קיצורי מקלדת

- `Ctrl + Shift + S` - התחל/עצור אימון
- `Ctrl + Shift + T` - הצג/הסתר תמלול
- `Ctrl + Shift + A` - פתח מחוון ביצועים
- `Ctrl + Shift + H` - הסתר הכל

## 🔧 פתרון בעיות מהיר

### לא שומע אותי?
1. בדוק הרשאות: `chrome://settings/content/microphone`
2. רענן את הדף
3. לחץ "Start Coaching" שוב

### אין תמלול?
1. פתח Console (F12) ובדוק שגיאות
2. המפתחות כבר מוגדרים בברירת מחדל
3. בדוק חיבור אינטרנט

### אין המלצות AI?
1. המתן 5-10 שניות - ה-AI צריך זמן
2. בדוק Console (F12) לשגיאות
3. המפתחות כבר מוגדרים
