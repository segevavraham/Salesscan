# 🎯 מאמן המכירות האולטימטיבי - מדריך מלא

## מה זה?

תוסף Chrome מתקדם שמשמש כמאמן מכירות אישי בזמן-אמת. התוסף מקליט את השיחה שלך, מתמלל אותה, מנתח אותה עם AI ונותן לך עצות מקצועיות במהלך הפגישה.

## ✨ מה חדש בגרסה 2.2

### 🚀 מנוע AI משופר
- **מאמן מכירות מומחה** - 25+ שנות ניסיון מקודד
- **תשובות בעברית** - כל ההמלצות בעברית מקצועית
- **זיהוי אותות קנייה בזמן-אמת** - מזהה כשהלקוח מעוניין
- **ניתוח התנגדויות מתקדם** - מזהה התנגדויות גלויות וסמויות
- **ניתוח פסיכולוגי** - מבין מה הלקוח חושב באמת

### 💎 UI מקצועי חדש לחלוטין
- **עיצוב RTL מושלם** - מותאם לעברית
- **התראות קריטיות** - מודיע כשיש אות קנייה או סיכון
- **כרטיס מאמן מקצועי** - מציג את כל המידע בצורה נוחה
- **צבעים דינמיים** - ירוק = אות קנייה, אדום = סיכון, כתום = הזדמנות

### 🎯 תכונות חדשות
- **התגובה המומלצת ביותר** - הצעת התגובה הכי טובה עם הסבר
- **אופציות חלופיות** - 3 אפשרויות תגובה שונות
- **שאלות מומלצות** - שאלות גילוי שכדאי לשאול
- **הערות המאמן** - מה אתה עושה טוב ומה לשפר
- **צעדים הבאים** - מה לעשות ב-30 השניות הבאות
- **אזהרות סיכון** - מתריע אם יש סיכון לאבד את העסקה

## 📦 התקנה מהירה

### שלב 1: הכנת API Keys

צריך 2 מפתחות API:

#### 1. AssemblyAI (תמלול)
1. היכנס ל: https://www.assemblyai.com/dashboard/signup
2. קבל 3 שעות חינם בחודש
3. העתק את ה-API Key

#### 2. OpenAI (AI Coaching)
1. היכנס ל: https://platform.openai.com/api-keys
2. צור API Key חדש
3. העתק אותו (שים לב: עולה כ-₪2-8 לשעת פגישה)

### שלב 2: התקנת התוסף

```bash
# התקנת תלויות
npm install

# בנייה
npm run build
```

1. פתח Chrome ועבור ל: `chrome://extensions/`
2. הפעל "Developer mode" (למעלה מימין)
3. לחץ "Load unpacked"
4. בחר את תיקיית `dist/`
5. התוסף מותקן!

### שלב 3: הגדרת מפתחות

1. לחץ על אייקון התוסף
2. בחר "Settings" או לחץ ימני → "Options"
3. הזן את שני המפתחות:
   - AssemblyAI API Key
   - OpenAI API Key
4. בחר שפה (עברית כברירת מחדל)
5. לחץ "שמור הגדרות"

## 💡 כרטיס המאמן - הסבר מפורט

### החלק העליון - מידע מהיר
```
┌────────────────────────────────────────┐
│  כיבוד   🟢  [X]                        │
└────────────────────────────────────────┘
```
- **שלב המכירה** - איפה אתה בתהליך
- **נורת סטטוס**:
  - 🟢 ירוק = הכל בסדר
  - 🟠 כתום = שים לב
  - 🔴 אדום = סיכון!

### ניתוח מהיר
```
┌─────────────┬─────────────┬─────────────┐
│ סנטימנט    │ מוכנות      │ מעורבות     │
│ חיובי       │ 8/10        │ 9/10        │
└─────────────┴─────────────┴─────────────┘
```
- **סנטימנט** - מצב הרוח של הלקוח
- **מוכנות להחלטה** - עד כמה הוא מוכן לקנות
- **מעורבות** - עד כמה הוא מעורב בשיחה

```
הלקוח מתעניין אבל מודאג מהמחיר ומהזמן ההטמעה
```
ניתוח פסיכולוגי של מה שעובר ללקוח בראש.

## 🔧 פתרון בעיות

### בעיה: "API keys חסרים"
**פתרון:**
1. לחץ ימני על התוסף → Options
2. הזן את שני המפתחות
3. לחץ שמור
4. רענן את הדף

### בעיה: המיקרופון לא עובד
**פתרון:**
1. בדוק הרשאות: `chrome://settings/content/microphone`
2. וודא שהאתר יכול לגשת למיקרופון
3. נסה ללחוץ "Start Coaching" שוב

### בעיה: אין תמלול
**פתרון:**
1. בדוק ב-Console (F12) אם יש שגיאות
2. וודא שה-AssemblyAI Key נכון
3. דבר חזק יותר
4. בדוק חיבור אינטרנט

### בעיה: אין המלצות AI
**פתרון:**
1. וודא שה-OpenAI Key נכון
2. בדוק ב-Console שגיאות 401/403
3. וודא שיש לך קרדיט ב-OpenAI
4. המתן כמה שניות - ה-AI צריך זמן

### בעיה: התוסף לא נטען באתרי פגישות
**פתרון:**
1. בדוק שהתוסף מופעל ב: `chrome://extensions/`
2. רענן את דף הפגישה (F5)
3. נסה לכבות ולהדליק את התוסף
